# pixel-agents-sentinel

This repository is a recreation of `https://github.com/pablodelucca/pixel-agents` for independent implementation tracking by Sovereign-Sentinel.

## Project Setup and Run Instructions

This project is a JavaScript/Node.js application.

1.  **Install dependencies (root and webview-ui):**
    ```bash
    npm install
    cd webview-ui && npm install && cd ..
    ```
2.  **Build the application:**
    ```bash
    npm run build
    ```
3.  **Run the application (VS Code Extension):**
    This project is a VS Code extension. To run, open the project in VS Code and press F5 to launch the Extension Development Host.
